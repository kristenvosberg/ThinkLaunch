API's


API stands for application program interface. Its a set of subroutine (meaning packaged, a unit) definitions, protocols, and tools used for bulding apps. Its open data, and is described by wikipedia as set of clearly defined methods of communication between various software components. A good API provides building blocks for how to run a computer program, assembled by the programmer.
